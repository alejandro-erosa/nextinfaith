## 🔧 CONCEPTOS GIT — Referencia rápida

```bash
git add .                          # Selecciona archivos modificados
git commit -m "descripción"        # Guarda cambios localmente con etiqueta
git push                           # Sube cambios a GitHub
```

**¿Cuándo hacer commit?** Cada vez que terminas algo que funciona.  
**¿El historial dónde vive?** En `.git` en tu disco local Y en GitHub después del push.  
**¿GitHub solo recibe el último estado?** No — recibe todos los commits acumulados desde el último push.
